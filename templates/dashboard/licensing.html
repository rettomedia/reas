API Dökümantasyonu: Lisans Doğrulama

Lisans doğrulama sisteminin entegrasyonu için gerekli adımlar.

1

Lisans Anahtarını Oluşturma

Panel üzerinden yeni bir lisans anahtarı oluşturabilirsiniz. Oluşturulan anahtarlar sadece Aktif durumdaysa geçerlidir.

2

API Endpoint

Lisans doğrulama için API endpoint'i:

POST https://your-domain/licensing/verify/
3

Request Gönderimi

API'ye gönderilecek veri JSON formatında olmalıdır:

{
"key": "LIC-XXXXXXXXXXXXXXX"
}
4

Response Yapısı

Başarılı bir doğrulama sonrası API aşağıdaki formatta döner:

{
"valid": true,
"message": "License is active.",
"license": {
"id": 1,
"key": "LIC-XXXXXXXXXXXXXXX",
"is_active": true
}
}

Geçersiz veya pasif lisans durumunda:

{
"valid": false,
"message": "Key is invalid."
}
5

Entegrasyon Önerisi

Müşteri uygulamaları HTTP POST isteği yapabilecek herhangi bir platform veya dilde hazırlanabilir:

Python
requests.post("https://your-domain/licensing/verify/", json={"key":"LIC-XXXX"})
JavaScript
fetch("https://your-domain/licensing/verify/", {method:"POST", body:JSON.stringify({key:"LIC-XXXX"})})
